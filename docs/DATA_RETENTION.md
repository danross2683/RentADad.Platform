# Data Retention Policy (Note)

This is a placeholder policy. Confirm with legal/compliance before production use.

## Retention

- Operational data (jobs, bookings, providers): retained indefinitely unless requested otherwise.
- Audit logs: retain for 12 months.
- Request logs: retain for 30 days.

## Deletion requests

- Requests must be logged and approved.
- Deletions should be permanent and auditable.
