# Lifecycle Diagrams Notes

These notes describe the intended diagrams to be created.

## Job

Draft -> Posted -> Accepted -> InProgress -> Completed -> Closed
Completed -> Disputed
Draft/Posted/Accepted -> Cancelled

## Booking

Pending -> Confirmed
Pending -> Declined
Pending -> Expired
Confirmed -> Cancelled

## Provider availability

Availability window created -> Bookable
Booking confirmed -> Blocked
Booking completed/cancelled -> Released (explicit)
